what we do

Clean Code

Clean code is a reader-focused development style that produces software that's easy to write, read and maintain. Developers are often tempted to consider their work complete when the application operates as expected. But we're not merely writing code for computer consumption. Clean code is about recognizing that your audience isn't just a computer, it's real-live humans!

responsive design

A web page should look good, and be easy to use, regardless of the device! Responsive Web Design is about using CSS and HTML to resize, hide, shrink, enlarge, or move the content so we can make it look good on any screen

tdd

Test-driven development(TDD) is a software development process that relies on the repetition of a very short development cycle: requirements are turned into very specific test cases, then the software is improved to pass the new tests, only.
